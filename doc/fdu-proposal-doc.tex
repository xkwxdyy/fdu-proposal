\chapter{前言}

\section{简介}

\cls{fdu-proposal} 是复旦大学研究生开题报告登记表 \LaTeX 模板，目前只开发了开题报告详情表部分，主要原因是系统中只需要上传这一页。

开题报告详情表是开题报告的主要内容，开发者是数学系学生，肯定会涉及到大量数学公式符号，以及参考文献，这些在 \LaTeX 中处理会更加得心应手，所以开发了本模板。
